## Deploy

### Deploy To Heroku Now:
[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/sunilroy007/ForceSub_Bot)

### Deploy in vps
- Clone this repo
```
git clone https://github.com/Jigarvarma2005/ForceSub_Bot
```
- Change directory
```
cd ForceSub_Bot
```
- Install requirements
```
pip3 install -r requirements.txt
```

### Configuration
Add [APP_ID](https://my.telegram.org/apps), [API_HASH](https://my.telegram.org/apps), [BOT_TOKEN](https://t.me/botfather) in [Config.py](Config.py) or in Environment Variables.

## Configs:
- Read `Config.py` to know all configs

### Deploying
- Run bot.py
```
python3 bot.py
```
